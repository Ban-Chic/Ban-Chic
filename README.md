# Ban:Chic
어떤 향수가 본인의 이미지 맞을지 모르시겠나요?
추구하고자 하는 이미지를 선택하여 설문을 제출하면 그에 맞는 향수를 추천해드립니다!

## 📆 프로젝트 기간
- 2024.03. ~ 2024.04 (6주)

## ✨️️ 주요 기능
1. 추구하고자 하는 이미지 기반 향수 추천
2. 옷차림 기반 향수 추천
3. 질문지 기반 향수 추천
